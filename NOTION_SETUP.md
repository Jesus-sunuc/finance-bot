# Notion API Setup - Local Development

## Setup Instructions

1. Copy `api/.env.example` to `api/.env`
2. Fill in your Notion credentials in `api/.env`
3. Share your Notion databases with the FinanceBot Integration
4. Run `docker-compose up --build`
